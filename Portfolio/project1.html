My Works

CRUD Operations

Developed a recipe application with full Create, Read, Update, and Delete functionalities. Users can add new recipes, view existing ones, update recipe details, and delete recipes as needed.

OMR Integration

Implemented Optical Mark Recognition (OMR) to enhance data input and processing. This feature allows users to easily input data using scanned forms or marked sheets, improving data accuracy and efficiency.

Advanced Queries

Utilized advanced querying techniques for efficient data retrieval and management. This includes complex SQL queries to manage recipe data effectively and integrate email sending functionalities for notifications and updates.
